# AI Content Automation System

An end-to-end AI system that transforms trending topics into ready-to-publish YouTube content.## 🧠 What this proves

This system is already capable of generating real AI-driven content outputs automatically.

It is not a mockup or demo idea.

## 🚀 Overview

This project builds an automated pipeline for YouTube content creation:

**Trend Detection → Script Generation → Voice Synthesis → Video Generation**

The goal is to reduce content production from hours to minutes using AI systems and automation workflows.

---

## 🧠 System Architecture

### 1. Trend Detection
- Collects trending topics from APIs / scraping / RSS feeds
- Filters high-engagement content ideas

### 2. Script Generation
- Uses LLMs (OpenAI / OpenRouter / open-source models)
- Converts topics into structured YouTube scripts

### 3. Voice Synthesis
- Converts scripts into natural speech using TTS models

### 4. Video Generation
- Combines audio + visuals into automated video outputs

---

## ⚙️ Tech Stack

- Python / Node.js
- n8n (automation layer)
- LLM APIs (OpenRouter / OpenAI / local models)
- TTS engines (ElevenLabs / open-source alternatives)
- FFmpeg (video rendering)
- Vercel (frontend demo)
